# Frames
Reference frame tree management as one header only.

Depends on [interpolation](https://github.com/ludodo23/interpolation).

# Install

Just copy both header files in your project:

```cpp
#include "interpolation.hpp"
#define FRAMES_WITH_EIGEN  // optional, enables the built-in Eigen backend
#include "frames.hpp"
```
